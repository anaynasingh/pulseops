## Burst Plan
**Goal:** Auto-ingest Teams meeting transcripts via Microsoft Graph (per-user delegated token), extract action items with GPT-4o into PROPOSED tasks, and surface them in a top-right dashboard bell where the user confirms/dismisses (with pre-add duplicate detection) to create real kanban tasks.
**CHARTER alignment:** Objective "Meeting Intelligence working: transcript upload ‚Üí action items, decisions, tasks extracted"; Success Criterion "Meeting transcript ‚Üí task extraction completes in <10 seconds"; Hard Spec "OpenRouter (GPT-4o) for all LLM inference" and "All schema changes via numbered migration files".
**Streams:** list (2 parallel, disjoint writers) ‚Äî Stream A Backend, Stream B Frontend. Backend is intentionally kept single-stream because every backend sub-concern must write `models.py` + `main.py` (shared writers), so splitting it would create coordination points rather than disjoint streams.
